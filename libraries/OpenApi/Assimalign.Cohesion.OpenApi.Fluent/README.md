# Assimalign.Cohesion.OpenApi.Fluent

A version-aware fluent authoring layer for the OpenApi model. Build `OpenApiDocument` graphs by chaining
builder members and nested callbacks, with version-gated fields diagnosed at authoring time.

- [docs/OVERVIEW.md](./docs/OVERVIEW.md) — purpose, scope, and usage
- [docs/DESIGN.md](./docs/DESIGN.md) — builder shape, version-awareness, boundaries
