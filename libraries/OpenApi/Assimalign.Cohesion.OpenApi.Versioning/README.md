# Assimalign.Cohesion.OpenApi.Versioning

Version targeting and transforms for the OpenApi model: convert a document between OpenAPI 3.0.4, 3.1.2,
and 3.2.0, applying the documented construct changes and reporting a diagnostic for every conversion and
every construct that cannot translate.

- [docs/OVERVIEW.md](./docs/OVERVIEW.md) — purpose, scope, and usage
- [docs/DESIGN.md](./docs/DESIGN.md) — conversion rules, the deep-copy strategy, and the shared capability contract
