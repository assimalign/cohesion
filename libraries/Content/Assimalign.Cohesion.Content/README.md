# Assimalign.Cohesion.Content

Root contracts of the Content library family: format-neutral content identity, format metadata,
stream access with explicit ownership, composition, and the parser/serializer seam that every
format package (binary containers, documents, text, media) implements.

- [docs/DESIGN.md](./docs/DESIGN.md) — contract model, stream-ownership rules, family layering
